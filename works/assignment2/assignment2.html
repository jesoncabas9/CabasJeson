CSS Positioning

CSS positioning allows you to control the position of an element in relation to its normal position,
its parent container, or the entire page. Here's a quick overview of the different positioning types:

Static

Elements are placed in the normal document flow (the default behavior).

Relative

The element is positioned relative to its normal position in the flow. You can adjust its position using top, right, bottom, or left, but it still takes up space in the document flow.

Absolute

The element is positioned relative to the nearest positioned ancestor (an element with position: relative, absolute, or fixed). If no such ancestor exists, it will be positioned relative to the initial containing block (usually the html element). It does not affect the layout of other elements and is removed from the document flow.

Sticky

The element toggles between relative and fixed positioning based on the scroll position. It's relatively positioned until it reaches a defined point during scrolling, at which it becomes fixed.
